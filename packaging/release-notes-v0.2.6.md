## v0.2.6

本版本增强了关键词与正则异常检测，并发布 Windows x64 单文件桌面程序。

### 新增与改进

- 桌面界面将普通关键词与正则表达式分栏输入，正则每行一条，无需添加 `re:` 前缀。
- 正则异常改为按实际命中值聚合：同一规则只有匹配到完全相同的内容，并出现在 2 个及以上公司组时才预警。
- 内置中国大陆手机号、中国大陆身份证、邮箱地址、中文地址四种正则预设，默认全部勾选，可逐项关闭。
- 报告和 `ai_summary.json` 增加实际重复内容与正则模式，便于 AI 和人工判断具体重复值。
- 旧配置中 `keywords` 的 `re:` 规则自动沿用新的同值预警语义，桌面加载时会迁移到正则输入框。
- Agent Skill 同步支持新配置和同值正则规则，继续保持自包含、轻量且不内置 PDF/OCR。

### Windows 版本

- `BidCheckSimilarity-v0.2.6-Windows-x64.exe`
- 单文件运行，包含 PDF 文本解析和扫描 PDF OCR 能力。

macOS 安装包将在 macOS 环境后续构建和发布。
